# ✨ Celestial Launcher Releases

Welcome to the official release repository of **Celestial Launcher** — the next-generation custom skin manager for League of Legends.

Designed for speed, stability, and a premium user experience, Celestial makes installing and managing custom skins easier than ever before.

---

## 🚀 Download

Get started in seconds:

1. Open the **Releases** section.
2. Download the latest **Celestial-Setup.exe**.
3. Install and launch Celestial.
4. Enjoy a seamless custom skin experience.
